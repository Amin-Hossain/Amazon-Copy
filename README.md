# 🛒 Amazon Clone

A responsive front-end clone of the Amazon.com homepage, built from scratch using **HTML5** and **CSS3**. This project replicates the core layout of Amazon's UI — including the navbar, category panel, hero banner, product grid, and footer.

🔗 **Live Demo:** [amazon-copy27.netlify.app](https://amazon-copy27.netlify.app/)

## ✨ Features

- Fully responsive Amazon-style **navbar** (logo, delivery location, search bar, account, cart)
- Category **panel** with quick links
- **Hero section** with banner image and smooth gradient blend into content
- **Product grid** with hover effects and aligned "See More" links
- Amazon-style **footer** with multi-column links, back-to-top button, and copyright section
- Smooth scroll-to-top functionality
- Built with pure **CSS Flexbox** for layout (no frameworks)

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox)
- [Font Awesome](https://fontawesome.com/) for icons

## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/amazon-copy.git

# Navigate into the project folder
cd amazon-copy

# Open index.html in your browser
```

No build tools or dependencies required — just open `index.html` directly in your browser.
